MHacks x Google AI Hackathon Project - GeminiScript

View on Devpost

This was a project that I created with 3 other teammates at the Mhacks x Google Hackathon. This was my first hackathon and I originally was going to do this solo, but I ended up hitting up a random girl (who seemed very experienced with hackathons) on slack and was able to join their team :). They already had this idea of essentially creating a google docs copilot that helped you use voice chat to link up with AI to edit and enhance your google doc. We used the Google Gemini API to connect to the LLM, and used python to link this with our AppScript, which edited the google doc. The point is that in the future, if we are using all this AR/VR stuff, then we won't be typing all that and would rather be using our voice, so this way we can make edits, fix errors, and create essays on google docs with just our voice. We didn't win anything but I was more than satisfied with my first hackathon experience. I met some super cool people from MSU and had a great time making a cool project. Check out the devpost to see the demo video!
